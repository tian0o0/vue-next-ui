## vue-next-ui

> 注意 ⚠️：功能开发中，还不可用！

> Attention⚠️：developing, inavailable!

### dev

```sh
npm run dev
```

### build

```sh
npm run build
```

### commit

> 不要使用 `git commit`

```sh
npm run commit
```

### release

> 打 tag、push 代码、发布 npm 包

> 执行此操作前需要 `npm login`，恢复设置 npm 源：npm set registry https://registry.npmjs.org

> 需在 master 分支下操作

```sh
npm run release
```

### changelog

> 根据 commit 记录自动生成 CHANGELOG.md

```sh
npm run changelog
```

### docs

```sh
docsify serve ./docs
```
